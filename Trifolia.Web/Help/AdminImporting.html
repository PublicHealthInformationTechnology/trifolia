Importing

Advanced Features ››

Trifolia allows data to be imported, but only by administrators (to ensure that the import does not affect the data in a way that is not intended). If you would like your data imported into Trifolia, please use the information below to format your import in a way that can be easily interpreted by Trifolia administrators, and send the Trifolia development team a request to import your data.
