This implementation guide is the second STU release of the third normative release of the HAI reporting templates. It revises one report, adds two new reports, revises one template, adds twelve new templates, revises four value sets, and adds two new value sets.

Twenty HAI report types are templated in this guide.

HAI Population Summary Reports:

  1. Antimicrobial Resistance Option (ARO) Summary Report (V2)
  2. Antimicrobial Use (AUP) Summary Report (V2)
  3. HAI Outpatient Procedure Component (OPC) Summary Report
  4. Healthcare Personnel Influenza Vaccination (HP-FLU) Summary Report
  5. Hemovigilance (HV) Summary Report (V2)
  6. Intensive Care Unit (ICU) Summary Report (V2)
  7. Neonatal Intensive Care Unit (NICU) Summary Report (V2)
  8. Prevention Process and Outcome Measures (POM) Summary Report (V2)
  9. Specialty Care Area (SCA) Summary Report (V2)
  10. Vascular Access Type Report (VAT) Summary Report (V2) HAI Single-Person Report Generic Constraints
  11. HAI AUR Antimicrobial Resistance Option (ARO) Report (V4)
  12. HAI Bloodstream Infection Report (BSI) (V2)
  13. HAI Central-Line Insertion Practice Numerator Report (V2)
  14. HAI Evidence of Infection (Dialysis) Report (V5)
  15. HAI Laboratory-Identified Organism (LIO) Report (V2)
  16. HAI Outpatient Procedure Component (OPC) Event Report
  17. HAI Procedure Denominator Report (V2)
  18. HAI Surgical Site Infection Report (SSI) (V2)
  19. HAI Urinary Tract Infection Numerator Report (UTI)
  20. Ventilator Associated Event (VAE) Report

Changes made in this release are summarized in the Appendix on High-Level Changes from Previous Releases. When new versions of templates appear for the first time in the implementation guide, Volumes 2, 3, and 4 of this guide contain detailed sections on “Changes from Previous Version”.
